# Predicción de la Progresión de la Diabetes
Este proyecto tiene como objetivo construir un modelo de regresión lineal para predecir la progresión de la diabetes a partir de información biomédica de los pacientes. Para ello, realicé un análisis exploratorio de los datos, la construcción del modelo de predicción y la evaluación de su desempeño utilizando Python, apoyándome en visualizaciones y métricas de regresión que facilitaron la interpretación de los resultados.
