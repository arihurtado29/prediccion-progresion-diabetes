# Predicción de la Progresión de la Diabetes
Este proyecto tiene como objetivo construir un modelo de regresión lineal para predecir la progresión de la diabetes a partir de información biomédica de los pacientes. Para ello, realicé un análisis exploratorio de los datos, la construcción del modelo de predicción y la evaluación de su desempeño utilizando Python, apoyándome en visualizaciones y métricas de regresión que facilitaron la interpretación de los resultados.

## Objetivo
Desarrollar un modelo de regresión lineal para predecir la progresión de la diabetes mediante el análisis de un conjunto de datos con variables biomédicas, con el propósito de conocer la relación entre las características de los pacientes y la variable objetivo; así como evaluar el desempeño del modelo desarrollado.

## Dataset
El dataset utilizado corresponde al conjunto de datos Diabetes disponible en la biblioteca scikit-learn. Este conjunto de datos contiene 442 registros y 10 variables biomédicas de pacientes, además de una variable objetivo que representa la progresión de la enfermedad un año después de las mediciones iniciales.
**Nota:** El conjunto de datos utilizados en este proyecto no se incluyen en el repositorio, ya que fue obtenido a través de la biblioteca scikit-learn.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Metodología
Durente el desarrollo del proyecto realicé el proceso de exploración y análisis del conjunto de datos. Primero cargué y revisé la información para conocer su estructura y las variables disponibles; posteriormente, preparé los datos para construir un modelo de regresión lineal y evalué su desempeño mediante diferentes métricas.
Finalmente, apliqué un proceso de selección de características para identidicar las variables más relevantes, reentrene el modelo y así comparar el desempeño del modelo obtenido.

## Resultados
El desarrollo del proyecto permitió construir un modelo de regresión lineal para estimar la progresión de la diabetes y analizar su desempeño mediantes métricas de evaluación y visualización. Además, la aplicación de un método de selección de características permitió identificar las variables con mayor relevancia para el modelo y comparar los resultados obtenidos.
